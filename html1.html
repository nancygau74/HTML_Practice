Some cheatsheet for CSS:

  • A, B: A & B both applies
  • A B: B can be any children of A
  • A > B: B has to be the direct child of A
  • A + B: B has to be the next element of A
  • :hover : change to style when mouse selected
  • :first-child: specifies for first child of element
  • :last-child: specifies for last child of element

    • **note: A & B as elements
